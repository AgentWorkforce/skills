# Running Headless Orchestrator

Skill for self-bootstrapping agent-relay infrastructure and autonomously managing a team of worker agents — no human intervention required.

## What it covers

- Starting the relay broker (foreground preferred)
- Verifying broker readiness before spawning workers
- Spawning and monitoring workers via CLI and MCP
- Lifecycle events and troubleshooting common issues

## Install

```bash
prpm install running-headless-orchestrator
```

See [SKILL.md](./SKILL.md) for the full guide.
